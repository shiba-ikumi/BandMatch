# BandMatch
A web app for assigning band part automatically.

〇概要

BandMatchは、吹奏楽サークルのパート決めを効率化するために開発したアプリです。
従来はリーダーが部員一人ひとりの希望や出席状況を確認しながら手作業でパートを決定しており、多くの時間を要していました。そこで、希望度や出席率などの情報をもとに、自動でパートを割り当てる仕組みを作成しました


〇データ仕様（重要）

本アプリでは、サークル独自の学年表記として、「08（1年）」「07（2年）」「06（3年）」「05（4年）」を用いています


〇開発背景

私は吹奏楽サークルに所属しており、演奏会ごとのパート決めに時間がかかっていることに課題を感じていました。
手作業では公平性の確保が難しく、担当者の負担も大きかったため、希望度や出席率を考慮して自動でパートを決定できるアプリの開発に取り組みました。

〇主な機能

・ 部員情報の登録

・ 希望パート（1st・2nd・3rd・Piccolo・曲に乗らない）の入力

・ 出席率の登録

・ 出席率40％以下の自動降り判定

・ Piccolo担当者の自動選出

・ パート人数制限を考慮した自動割り当て

・ CSVへのデータ保存

・ DataTableによる結果表示

〇使用技術

・ Python
・ Flet
・ CSV
